## Briefing

O site se trata de um portifólio . 
Este __Briefing__ trata-se do desenvolvimento desse portifólio, como atividade avaliativa de Design de Interface e IHC.
Nome do autor: **Luis Gustavo Tabile**.

## Linguagens

O site se baseia em HTML5, utilizando tags padrões. Possui JavaScript para rodar a lógica de alguns menus.
A estilização foi realizada utilizando Bootstrap e CSS3.

## Desenvolvimento

O portifólio foi desenvolvido com objetivo de ser o mais minimalista possível, utilizando tons escuros para estar de acordo com o pedido do cliente.
Foi optado pela clareza de informação com poucas __"distrações na tela"__ de forma em que o usuário possa acessar o que deseja da forma mais fácil e rápida possível.
Possui tempo de carregamento super rápido pois não tem nenhum arquivo pesado, facilitando até para quem não possui uma conexão com a Internet muito boa.
